# 18 - API REST de Productos

Descripción
- API REST simple para gestionar productos (CRUD) usando Node.js y Express.

Tecnologías
- Node.js, Express

Cómo ejecutar
1. Abrir una terminal en la carpeta del proyecto.
2. Ejecutar `npm install` para instalar dependencias.
3. Ejecutar `npm start` para iniciar el servidor en `http://localhost:3001`.

Endpoints
- `GET /api/products`
- `GET /api/products/:id`
- `POST /api/products` (JSON en body)
- `PUT /api/products/:id`
- `DELETE /api/products/:id`
